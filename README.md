# 3D Tic-Tac-Toe with Gravity

A 4×4×4 three-dimensional tic-tac-toe game with gravity mechanics. Pieces drop to the lowest available position in each column, like Connect Four but in 3D. Get four in a row in any direction—horizontal, vertical, or diagonal across any plane—to win.

Play against a friend locally or challenge one of four AI bots with varying difficulty levels.

## Development Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

```bash
npm run build
npm run preview
```
